# roblox-user-checker
safe script just over 200 lines! runs using python. input ur username options & run :)

___before you run___:
make a **REASONABLE** sized list of usernames in usernames.txt. if its too large for your device/network to handle, you will be prompted with an error
make sure you have python installed from the official website! no extra packages are needed for this script

___how to run___
it should work with any command center you use, but i've only tested with windows powershell. 
command to run: python checker.py 

if the command doesn't run for you, try to input the exact file location. if this fails, it is most likely internet related
